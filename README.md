# 🎯 Guess The Number Game

A fun and interactive Guess The Number Game developed using HTML, CSS, and JavaScript. Players choose a difficulty level and try to guess a randomly generated number within a limited number of attempts. The game provides helpful hints, tracks the player's score, and displays a result page at the end of each game.

## ✨ Features

- 🏠 Home page with difficulty selection
- 🎮 Three difficulty levels:
  - Easy (1–20)
  - Medium (1–50)
  - Hard (1–100)
- 💡 Hint system
- 🔢 Limited attempts based on difficulty
- 📊 Live score tracking
- ⬆️ Too High / ⬇️ Too Low feedback
- 🏆 Separate result page
- 🔄 Play Again and Home buttons
- 📱 Responsive and user-friendly design
- 🎨 Fixed colorful background with modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.
3. Select a difficulty level and click Start Game.
4. Guess the correct number using the provided hints before running out of attempts.
